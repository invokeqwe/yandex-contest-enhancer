# Yandex Contest Enhancer

Chrome extension that adds a dark theme, syntax highlighting, and UI improvements for Yandex Contest.

## Features

- Dark mode toggle for Yandex Contest pages.
- CodeMirror syntax highlighting for the solution editor.
- Cleaner problem statement, samples, submissions, and standings UI.
- Better contrast for verdicts, messages, tags, tables, and controls.

## Installation

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click "Load unpacked".
4. Select this project folder.
